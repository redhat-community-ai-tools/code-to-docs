# ARCHIVE Documentation Index

## Overview
This documentation area contains archived information about the legacy v1 `Processor`. It serves as a reference for users who are still operating on version 1.x of the software. The purpose of this archive is to provide visibility into older methods and functionalities that are no longer supported in the latest version, ensuring that users can maintain legacy systems while encouraging a transition to v2.

## Files Summary
- **archive\v1.md**: This file outlines the methods available in the legacy v1 `Processor`, specifically detailing the `run()` method that processes all queued items. It notes that this documentation will not receive further updates, guiding users to switch to version 2 for new projects.
